# AutoTest
[功能描述]：接口自动化测试框架，Excel管理测试用例数据，HTMLTestRunner生成测试报告，测试完成后邮件发送测试报告，jsonpath方式做预期结果数据处理，后期多样化处理，CI集成...
[开发环境]：python3 + unittest + requests + HTMLTestRunner
[项目结构]：
 AutoTest
    |--common   公共的py文件、工具类
    |--config   配置文件 
    |--data   excel管理的测试用例数据
    |--logs   日志文件
    |--report   测试报告
    |--testcase   测试用例
    run_case    执行用例的py文件
